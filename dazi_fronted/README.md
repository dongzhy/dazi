## 一、项目介绍

基于 Vue3 + Vite 开发，采用 script setup 语法糖，搭配 Vant 移动端组件库 实现全移动端适配。
项目采用前后端分离架构，配合后端 Spring Boot 服务，完整实现用户登录注册、个人资料编辑、标签筛选、智能兴趣匹配、队伍创建 / 加入 / 管理等核心社交业务。封装统一网络请求、路由管理、全局交互逻辑，代码分层清晰，适合 Vue3 移动端实战学习与二次开发。
## 二、技术栈
- 核心技术
- 核心框架：Vue3（Composition API + script setup）
- 构建工具：Vite
- 路由管理：Vue Router
- 网络请求：Axios（请求 / 响应拦截、统一异常处理）
- UI 组件库：Vant 4 移动端组件库
### 辅助工具 & 语法
- qs：URL 参数序列化，适配后端数组参数接收规则
- 原生 CSS + scoped：组件样式隔离
- 路由监听、动态路由、路由传参
- Vue 内置响应式：ref / computed 管理页面状态
## 环境依赖
### 运行环境
- Node.js：16.x 及以上版本
- 包管理器：npm /yarn/pnpm
### 快速启动
1. 前置准备
   本地安装好 Node.js 环境
   优先启动后端服务，前端所有接口依赖后端，否则无法正常请求数据
2. 安装项目依赖
   进入前端项目根目录，执行以下命令：
   bash
   运行
npm install

3. 启动开发环境
   bash
   运行
   npm run dev
   本地默认访问地址：http://localhost:5173
   支持热更新，修改代码实时预览效果
4. 项目打包（生产环境）
   bash
   运行
   npm run build
   打包完成后，静态资源输出至根目录 dist 文件夹，可直接部署到 Nginx 等静态服务器。
## 三、核心功能模块
  ### 3.1 全局布局与路由
   #### 全局布局
   - 顶部导航栏：返回按钮、动态页面标题、搜索快捷入口
   - 底部 Tab 栏：首页、队伍、个人中心三大模块快速切换
   #### 路由能力
   - 统一路由表管理，监听路由变化自动更新页面标题
   - 支持路由传参：标签、编辑字段、队伍 ID 等参数传递
   - 路由切换时自动重新加载对应页面数据
 ###  3.2 用户模块
   - 登录页面：账号密码登录，登录成功自动跳转到来源页面，登录态本地缓存
   - 个人中心：展示用户基础信息，提供资料编辑、我的队伍等功能入口
   - 资料编辑：支持修改昵称、头像、性别、手机号、邮箱、兴趣标签
   - 数据格式适配：前端标签数组 ↔ 后端 JSON 字符串 双向转换，兼容后端存储格式
 ###  3.3 首页 & 智能匹配
   普通推荐：调用后端推荐接口，加载推荐用户列表（依托后端 Redis 缓存加速）
   心动模式：开关切换至标签匹配模式，调用后端编辑距离算法，匹配高相似度用户
   用户卡片：统一封装组件，展示头像、昵称、星球编号、个人简介、兴趣标签
 ###  3.4 标签搜索模块
   树形标签选择器，支持关键词检索筛选标签
   多选标签后提交查询，后端根据标签交集筛选对应用户
   空数据提示，无搜索结果时展示友好占位页面
 ###  3.5 队伍模块
   队伍类型区分
   - 公开队伍：无需密码，直接加入
   - 加密队伍：弹出密码输入弹窗，验证密码后方可加入
   - 私有队伍：前端做权限拦截，普通用户无法加入
   #### 核心功能
   - 创建队伍：配置队伍名称、描述、人数上限、过期时间、队伍类型与密码
   - 队伍管理：修改队伍信息、解散队伍、退出队伍、队长自动转让
   - 队伍列表：区分「我创建的队伍」和「我加入的队伍」，展示队伍人数、过期时间等信息
## 亮点
   - 组件化开发：公共卡片、布局抽分为独立组件，代码复用率高、易维护
   - 请求分层管理：Axios 拦截 + 接口封装，统一处理请求、异常、状态码
   - 纯移动端适配：基于 Vant 原生移动端组件，贴合手机端交互习惯
   - 完善异常处理：网络错误、接口报错、参数非法、登录失效等场景全覆盖
   - 分层架构清晰：页面、组件、接口、状态职责分离，便于后续迭代扩展
   - 路由规范：动态标题、路由监听、路由传参等功能规范化使用
